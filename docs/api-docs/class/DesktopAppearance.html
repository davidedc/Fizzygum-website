Instance Method Details

# (void) paintIntoAreaOrBlitFromBackBuffer(aContext, clippingRectangle, appliedShadow)

This method only paints this very morph i.e. it doesn't descend the children recursively. The recursion mechanism is done by fullPaintIntoAreaOrBlitFromBackBuffer, which eventually invokes paintIntoAreaOrBlitFromBackBuffer. Note that this morph might paint something on the screen even if it's not a "leaf".
